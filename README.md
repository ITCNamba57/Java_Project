<h2><a id="user-content-minesweeper-game" class="anchor" href="#minesweeper-game" aria-hidden="true"><span class="octicon octicon-link"></span></a>Minesweeper game</h2>

<h3><a id="user-content-rules" class="anchor" href="#rules" aria-hidden="true"><span class="octicon octicon-link"></span></a>Rules</h3>

<p>You can see a minefield with many square cells in it.</p>

<p>Your goal is to find all mines and not detonate any.</p>

<p>You can open cell by left-clicking on it.</p>

<ul>
<li>If cell is empty, you keep playing</li>
<li>If there is mine in this cell, "Game Over" and You lose.</li>
<li>If there is number in the cell - it shows how many mines are in the eight surrounding cells</li>
</ul>

<p>If You suspect there is mine in closed cell - You can right-click on this cell to mark it. </p>

<p>It will change color to red.</p>

<p>If You change Your mind - right-click this cell again and it will change its color back.</p>

<p>Above minefield You can see number of marked cells and number of planted mines.</p>

<p>If You will open all cells without mines - You will win!</p>
